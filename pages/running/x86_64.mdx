# Live boot dahliaOS on x86_64 systems

dahliaOS supports a wide range of devices, this guide will tell you how to flash it on a USB and what hardware you need.

## Requirements

<Callout type="warning" emoji="⚠️">
  NVIDIA graphics won't work well, use onboard graphics for the best result.
</Callout>

- **4 GB of RAM**
- **64 bit** quad-core processor with **Intel HD graphics**.

## Flash the ISO

<Callout type="warning" emoji="⚠️">
  There are two ISOs to choose from, **EFI** and **Legacy**. In case you have an
  older device, flash the .ISO file instead of the .zip.
</Callout>

1. Download the latest release [here](https://dahliaos.io/download).
1. Download [Etcher](https://www.balena.io/etcher/).
1. Flash the .ISO file in case you have an older device **or** the .zip in case you have a newer one.
1. Reboot your system and go in your UEFI, go to the **Boot** tab and select the USB.
1. Your system should boot into dahliaOS.
